# MindFlow — UX/UI Case Study

## Descripción
Proyecto de diseño UX/UI enfocado en una app mobile para el registro y seguimiento de emociones.  
Se trabajó el proceso completo desde investigación de usuarios hasta prototipo interactivo en Figma.

## Problema
Las personas que desean registrar sus emociones suelen abandonar aplicaciones complejas o poco intuitivas. Era necesario diseñar una experiencia clara, accesible y rápida de usar.

## Solución
Se diseñó una aplicación con flujos simples, jerarquía visual clara y sistema de componentes reutilizables.  
Se desarrollaron:
- investigación de usuarios  
- user persona  
- journey map  
- arquitectura de información  
- wireframes  
- prototipo de alta fidelidad  

## Resultado
Se obtuvo un prototipo funcional que permite registrar emociones de forma rápida y visualizar patrones.  
El proyecto demuestra un proceso UX completo y centrado en el usuario.
